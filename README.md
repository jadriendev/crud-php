# CRUD System

A simple and fully functional **CRUD (Create, Read, Update, Delete) system** built with **PHP**, **MySQL**, and **Tailwind CSS**. This project demonstrates basic database operations and web application design with a modern, responsive interface.

## Features

- **Create**: Add new users with name and email.  
- **Read**: View all users in a clean, sortable table.  
- **Update**: Edit existing user details easily.  
- **Delete**: Remove users with a confirmation popup.  
- **Responsive Design**: Built with Tailwind CSS for a mobile-friendly interface.  
- **Notifications**: Toast-style success messages for adding and updating users.  
- **Role Detection**: Optional role-based display (Admin / User).  

## Technologies Used

- **HTML** – Page structure and Forms
- **PHP** – Backend logic and CRUD operations  
- **MySQL** – Database to store user data  
- **Tailwind CSS** – Modern, responsive frontend  
- **Font Awesome** – Icons for UI elements
